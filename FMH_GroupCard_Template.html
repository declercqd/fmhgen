ℹ️ Format du fichier CSV

Le fichier CSV doit respecter la structure suivante :

Ligne Contenu Exemple
1 Titre du fichier (ignoré) FMH_Joueurs_Modele
2 Compétition;valeur Compétition;WC Indoor
3 Lieu;valeur Lieu;Nottingham
4 Dates;valeur Dates;26-31 Mars 2026
5 En-têtes colonnes prénom;nom;numéro;poste;Catégorie
6+ Un joueur par ligne Damien;DECLERCQ;7;Défenseur;M45

Séparateur : point-virgule ( ; )
Encodage : UTF-8 recommandé (pour les accents)
Catégorie : renseignée par joueur (colonne 5) — auto-remplit le champ Catégorie si unique
Lignes 2-4 : remplissent automatiquement le Sous-titre (Compétition · Lieu · Dates)

Mode d'emploi :
